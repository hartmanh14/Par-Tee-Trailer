# Par-Tee Trailer

Internal launch package for a golf-first mobile simulator trailer business based in Troy, Ohio / Dayton metro.

## Current Direction

- Launch first as a mobile golf simulator trailer.
- Default setup: group golf lounge with built-in participant seating.
- Fundraiser setup: seating swapped for sign-in desk / management hub.
- Fundraiser competitions: closest-to-the-pin and longest-drive.
- Movie and gaming configurations are Phase 2 only unless intentionally reactivated.

## Start Here

- `project_hub/index.html` - internal project-management hub.
- `draft_website/index.html` - customer-facing draft website.
- `PROJECT_STATUS.md` - current status and next actions.
- `LLM_HANDOFF_CONTEXT.md` - context for another LLM to continue the work.
- `RENDER_DEPLOYMENT.md` - Render Blueprint deployment instructions.

## Key Files

- `02_90_day_execution_tracker.md` - 90-day validation/build checklist.
- `03_local_market_validation_worksheet.md` - Troy/Dayton competitor and market worksheet.
- `04_lean_budget_break_even_model.xlsx` - editable budget and break-even model.
- `05_vendor_shortlist_and_outreach.md` - trailer, simulator, and wrap vendor outreach.
- `08_ohio_legal_insurance_licensing_checklist.md` - Ohio legal/insurance/licensing checklist.
- `09_gap_analysis_and_solutions.md` - gaps, risks, and fixes.
- `shareable_photos/` - phone-friendly JPEG concept images.

## Important

The `project_hub` site is internal only. Do not treat it as the customer-facing website.

- `par-tee-trailer-customer-site` serves `draft_website`.
